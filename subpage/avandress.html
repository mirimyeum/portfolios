Avandress Responsive Web Design

본 프로젝트는 온라인 쇼핑을 선호하는 소비자가 Avandress의 정보, 혜택, 의류를 다양한 디바이스에서 쉽게 확인할 수 있도록 제작한 반응형 웹디자인입니다.
브랜드 리뉴얼을 위해 프로젝트 소개, 벤치마킹, 정보 설계, 스타일가이드 등을 포함한 컨셉 보고서를 먼저 작성하였고, 이를 기반으로 PC·Tablet·Mobile 환경에 최적화된 와이어프레임과 UI 디자인을 구현했습니다.
사이트 특성에 맞는 이미지를 선별·활용하였으며, 반복되는 스타일 요소는 원본 컴포넌트로 관리하여 수정과 유지보수에 효율적인 디자인을 완성했습니다.
본 작업은 디자인 레퍼런스 조사부터 시각 디자인까지 전 과정을 1일간 단독 수행하였습니다.

  • Target Analysis

    • 연령층: 20~30대 온라인 쇼핑에 익숙한 소비자
    • 특징: 트렌디한 패션에 관심이 많고 모바일 중심의 쇼핑 경험 선호
    • 디지털 사용 습관: 모바일 웹·앱을 통해 브랜드와 제품 탐색 / 인스타그램·SNS 피드를 통한 최신 트렌드 확인
    • 니즈: 직관적이고 편리한 인터페이스, 반응형으로 최적화된 쇼핑 환경, 브랜드 신뢰성

    Concept Description

    • 본 디자인의 핵심은 브랜드 아이덴티티를 살리면서 직관적이고 일관된 사용자 경험을 제공하는 것입니다.
    • 스타일가이드 기반의 통일된 컬러·타이포그래피 적용
    • 다양한 디바이스에서 반응형으로 자연스럽게 전환되는 레이아웃 설계
    • 와이어프레임 → 시각 디자인 과정에서 Avandress 특유의 모던하고 세련된 분위기 강조

  • Final Outcome

    • Avandress 반응형 웹사이트 UI 디자인 (PC, Tablet, Mobile)
    • 브랜드 톤앤매너를 반영한 레이아웃 & 스타일가이드
    • 시각적 일관성을 유지하는 컴포넌트 기반 UI 구조
    • 온라인 쇼핑 경험을 강화한 직관적인 사용자 흐름 설계

    Tools & Skills Used

    • 디자인 툴:
      Figma, Adobe Photoshop
    • 스킬:
      - 컨셉 보고서 작성 (프로젝트 소개, 벤치마킹, 정보 설계, 스타일가이드 정리)
      - 반응형 UI/UX 디자인 (PC·Tablet·Mobile 최적화)
      - 와이어프레임 및 스타일가이드 제작
      - 브랜드 리뉴얼 및 톤앤매너 반영
      - 컴포넌트 기반 효율적 디자인 관리
      - 이미지 리서치 및 시각 자료 선별
