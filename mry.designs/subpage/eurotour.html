EuroTour Responsive Web Design

본 프로젝트는 유럽 여행을 계획하는 모든 여행자를 대상으로 브랜드 아이덴티티와 반응형 웹사이트를 디자인한 작업입니다.
타겟 고객이 직관적으로 브랜드를 이해하고 편리하게 정보를 탐색할 수 있도록 로고, 컬러 시스템, 웹 UI를 설계하고, 명확한 정보 구조를 적용했습니다.
총 5일간 와이어프레임 제작부터 시각 디자인까지 전 과정을 담당했습니다.

  • Target Analysis

    • 연령층: 20~40대 여행자
    • 특징: 여행 일정 계획과 예산, 관광지 정보를 한 번에 확인하고 싶어 하는 사용자
    • 디지털 사용 습관: 모바일·태블릿·PC 등 다양한 기기를 활용해 여행 정보를 탐색
    • 니즈: 직관적인 UI, 감각적인 비주얼, 쉽고 빠른 정보 접근

    Concept Description

    EuroTour는 유럽 여행의 설렘과 다양성을 시각적으로 표현하는 것을 목표로 합니다.
    블루계열 톤의 포인트 컬러 팔레트와 간결한 타이포그래피를 사용하여, 사용자에게 편안하고 친근한 첫인상을 제공합니다.
    반응형 디자인을 적용해 모바일과 데스크톱 환경 모두에서 일관성 있는 경험을 제공하며, 정보 접근성을 극대화했습니다.

  • Final Outcome

    • 브랜드 로고 디자인: 유럽 여행의 상징성을 반영한 심플 로고 제작
    • 컬러 시스템: 여행의 다양성과 따뜻함을 담은 블루톤 중심의 팔레트
    • 반응형 웹 UI 디자인: 다양한 기기에서 최적화된 화면 구성
    • 정보 구조 개선: 여행지·숙소·후기·문의 정보를 한눈에 확인 가능하도록 메뉴 구성

    Tools & Skills Used

    • 디자인 툴:
      Figma, Adobe Photoshop
    • 스킬:
      - UI/UX 디자인
      - 반응형 웹 디자인
      - 스타일가이드 제작
      - 와이어프레임 설계
