Popup Design

본 프로젝트는 모바일 상품 구매 시, 구매자들이 더 많은 혜택을 받을 수 있도록 안내하는 팝업창을 디자인한 작업입니다.
타겟 고객이 Call To Action을 자연스럽게 실행할 수 있도록, 혜택과 정보를 직관적이면서도 흥미롭게 전달하는 컬러 시스템, 일러스트 요소, UI 컴포넌트를 적용하였습니다.
또한 한눈에 이해할 수 있는 정보 구조를 설계하여 사용자의 참여율을 높였습니다.
총 2일간 디자인 레퍼런스 조사부터 시각 디자인까지 전 과정을 담당했습니다.

  • Target Analysis

    • 연령층: 20~40대 모바일 쇼핑 이용자
    • 특징: 한정 혜택·이벤트에 반응이 빠르고, 시각적 요소에 민감
    • 디지털 사용 습관: 모바일 쇼핑 앱과 웹에서 이벤트 배너·팝업에 자주 노출
    • 니즈: 직관적인 정보 전달, 즉시 참여 유도, 시각적으로 매력적인 혜택 표현

    Concept Description

    팝업창은 즉각적인 시선 집중과 간결한 정보 전달을 핵심 컨셉으로 삼았습니다.
    메인 컬러는 각 앱의 톤 팔레트를 반영하여, 이벤트와 혜택을 연상시키는 초록·핑크 톤을 적용했습니다.
    여기에 일러스트와 아이콘을 더해 사용자가 내용을 직관적으로 이해하고, 한 번 더 시선을 머물게 하도록 구성했습니다.
    또한 버튼과 강조 영역을 아래로 배치해서 시각적으로 구분해 Call To Action 클릭률을 극대화했습니다.

  • Final Outcome

    • 팝업 UI 디자인: 모바일 화면에 최적화된 레이아웃과 정보 구조
    • 컬러 시스템: 시선을 끄는 메인 컬러와 혜택 강조를 위한 포인트 컬러
    • 일러스트 요소: 혜택 아이콘, 간단한 그래픽 장식으로 분위기 강화
    • UI 컴포넌트: 명확한 버튼, 강조 배너, 시각적 계층 구조 적용

    Tools & Skills Used

    • 디자인 툴:
      Figma, Adobe Photoshop, Adobe Illustrator
    • 스킬:
      - UI 디자인
      - 팝업 UI 구성
      - 정보 구조 설계
      - 스타일가이드 제작
      - 그래픽 요소 제작
      - 디자인 리서치 & 레퍼런스 분석 역량
      - 이미지 리서치 및 선별
