Planta Mobile APP UIUX Design

본 프로젝트는 식물 키우기를 사랑하는 소비자가 Planta 브랜드의 모바일 앱을 통해 식물 정보를 쉽게 습득하고, 구매까지 안전하게 진행할 수 있도록 UI·UX를 설계한 작업입니다.
자연과 조화를 이루는 버튼, 색상, 타이포그래피, 레이아웃 등 UI 요소를 적용하였으며, 사용자 플로우와 편의성을 고려해 전반적인 앱 경험을 최적화했습니다.
총 5일간 디자인 레퍼런스 조사부터 시각 디자인까지 전 과정을 단독으로 담당했습니다.

  • Target Analysis

    • 연령층: 20~40대 식물 애호가, 홈가드닝 관심층
    • 특징 식물 정보, 희귀 식물 구매, 인테리어 소품에 관심도 높음
    • 니즈: 모바일 앱을 통해 식물 관리 팁과 다양한 식물 종류를 확인 후 온라인 구매 진행, 추후 ‘나의 식물’ 등록 후 관리 가능

    Concept Description

    • 핵심 컨셉:
      “자연과의 연결, 손끝에서 완성”
    • UI 방향:
      - 식물의 생동감을 느낄 수 있는 그린 계열 메인 컬러와 부드러운 톤의 보조 색상 사용
      - 초보자도 직관적으로 조작 가능한 단순화된 버튼, 아이콘, 메뉴 구조 적용
    • UX 방향:
      - 로그인 → 인기식물 추천 및 나의 식물 정보 확인 → 정보 탐색 → 식물 선택 → 구매 → ‘나의 식물’ 등록 → 관리까지 연속적인 플로우 설계
      - 불필요한 단계를 최소화하고, CTA(Call To Action) 버튼을 시각적으로 강조하여 구매 전환율을 높임
    • 디자인 포인트:
      - 식물 사진 중심의 비주얼 레이아웃
      - 한눈에 카테고리와 추천 제품을 확인할 수 있는 홈 화면
      - 관리 기능 페이지에 직관적인 상태 표시와 편리한 추가 기능 적용

  • Final Outcome

    • 프로토타입 제작이 가능한 모바일 앱 UI 시안
    • 카테고리별 식물 정보 상세 페이지 디자인
    • 장바구니 및 결제 플로우 UI 설계
    • 브랜드 톤 앤 매너에 맞춘 컬러 팔레트와 타이포 시스템 구축

    Tools & Skills Used

    • 디자인 툴:
      Figma
    • 스킬:
      - UI 요소 설계 (색상, 이미지, 네비게이션 바, 타이포그래피, 그림자 효과, 버튼, 아이콘, 레이아웃)
      - UX 플로우 설계
      - 스타일가이드 시스템 개발
      - 타이포그래피 리서치 및 선정
      - 디자인 레퍼런스 조사 및 분석
      - 이미지 리서치 및 선별
